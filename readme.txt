Adds capability to filter WooCommerce products based on featured status.

== Description ==

Improves WooCommerce back-end by adding capability to filter products based on featured status.

== Installation ==

Install plugin via the WordPress.org plugin repository or by uploading the files to your server.

== Frequently Asked Questions ==

= Will it work for other post types? =

No, it works only for products list on WooCommerce back-end.

= Will it cause errors if WooCommerce is inactive? =

No. Functionality is loaded only when WooCommerce is installed and active. You don't have to deactivate this plugin if you have deactivated WooCommerce plugin for some maintenance or debugging tasks.
